08 // Accordion / FAQ

Comment fonctionne le systeme ? expand_more
Le systeme utilise une architecture moderne avec des composants reutilisables et un design system coherent base sur la palette Lavande Dreams.
Quelles sont les technologies utilisees ? expand_more
Tailwind CSS pour le styling, Material Symbols pour les icones, et Space Grotesk pour la typographie.
Comment personnaliser les couleurs ? expand_more
Modifiez les variables CSS dans le fichier de configuration ou directement dans le tailwind.config pour adapter les couleurs a votre projet.
