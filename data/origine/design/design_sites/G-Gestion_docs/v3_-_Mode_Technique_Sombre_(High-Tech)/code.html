2.1 Hardware Requirements

calendar_today Oct 24, 2023 person_edit Engineering Dept lock Restricted (L3)

Core Processing Unit

The central processing cluster MUST utilize a minimum of AMD EPYC™ 7763 architecture or equivalent high-performance computing units. Each node requires a specialized cooling manifold to handle a thermal design power (TDP) exceeding 280W during peak inference loads.

Cores

64 Physical

Cache

256MB L3

Clock Speed

3.5 GHz

Fig 2.1.A - Interconnect Topology Scale: 1:12

Memory Subsystem

System memory must consist of DDR5 ECC-Ready DIMMs with a minimum transfer rate of 4800MT/s. Data integrity is paramount; any hardware non-compliance with parity check requirements will trigger an immediate Level 4 failure alert in the diagnostic hub.

warning

Compliance Note

Mixing memory brands within a single channel is strictly prohibited and voids Tier-1 system warranties under ISO-9001 protocols.
